# University Attendance Management System

This project is a web-based Attendance Management System designed for University to automate and simplify the process of tracking student attendance.

The system replaces traditional manual attendance methods with a digital solution that allows professors, students, and administrators to manage attendance data efficiently and in real time.

It provides role-based access and includes features such as attendance session management, student dashboards, statistical reporting, and absence justification handling.

## Key Features

* Role-based authentication (Admin, Professor, Student)
* Attendance session creation and management
* Student attendance tracking and history
* Justification request system for absences
* Administrative dashboard with statistics and charts
* Import/export of student lists (Progres format)

## Technologies Used

* PHP (Backend)
* MySQL (Database)
* HTML, CSS, JavaScript (Frontend)
* jQuery (Frontend interactions)

## Project Goals

* Automate attendance tracking in universities
* Improve accuracy and reduce manual errors
* Provide real-time access to attendance data
* Support reporting and analytics for administrators
